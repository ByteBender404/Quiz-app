# 🧠 Neural Net Quiz Arena

A real-time, full-stack multiplayer quiz application powered by an AI generation engine. Users can challenge their tech knowledge in solo training modules or enter a synchronized live multiplayer arena.

## 🚀 Live Demo
* **Frontend:** [https://quiz-arena-steel.vercel.app](https://quiz-arena-steel.vercel.app)
* **Backend API:** [https://quiz-arena-backend-9s6p.onrender.com](https://quiz-arena-backend-9s6p.onrender.com)

## ⚡ Features
* **AI Quiz Generation:** Instant custom assessment modules built via the Google Gemini API engine.
* **Real-time Multiplayer:** Synchronized lobby state and live score tracking using WebSockets (`Socket.io`).
* **Cyber-Athletic Dashboard:** Detailed player statistics tracking, including Total Quizzes Played, Win Rate, and Player XP progression.
* **Secure Authentication:** JWT-based user authentication sessions.

## 🛠️ The Tech Stack
* **Frontend:** React, TypeScript, Tailwind CSS, Vite
* **Backend:** Node.js, Express, Socket.io
* **Database:** MongoDB (via Mongoose)
* **Deployment:** Vercel (Frontend) & Render (Backend)
